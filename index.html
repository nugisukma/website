Lesson Learned

BPMN

A standard for modeling business processes in the form of diagrams that are easily understood by various stakeholders, both technical and non-technical.Some important elements to learn in BPMN include:Event, Activity,Gateway , Flow, Pool and Lane

HTML

the standard markup language used to create and structure content on web pages. It provides the foundation for web development, allowing developers to design the layout and organize content like text, images, links, and interactive elements.

DFD

a modeling tool used to illustrate how data flows through a system, whether in the context of software development, information systems, or business processes. DFD helps in understanding the flow of data, the sources and destinations of that data, and the processes that manipulate the data.

ERD

visual tool used in database design to describe the relationships between entities or objects in a system. ERD makes it easier to understand how data interacts with each other in a system, and is used to create a structured and efficient database design
